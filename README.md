 # SQL-Projects
Analyzed Ola ride booking data using SQL to identify trends in customer behavior, revenue, and ride performance. Used queries with filtering, grouping, and aggregation functions to find insights like peak booking hours, payment preferences, vehicle performance, and ride cancellations.
# Ola Ride Booking Data Analysis (SQL Project)

## 📌 Objective
To analyze ride booking data and extract insights related to customer behavior, revenue trends, and ride patterns.

## 🛠 Tools Used
- SQL (MySQL)

## 📊 Key Analysis
- Total revenue from completed rides
- Peak booking hours
- Payment method analysis
- Vehicle type performance
- Ride cancellation analysis

## 📈 Insights
- UPI is a commonly used payment method
- Peak bookings occur during specific hours
- Certain rides have higher cancellation rates

## 📂 Files
- ola_project.sql → SQL queries used in the project

## 🚀 Conclusion
This project demonstrates how SQL can be used to analyze real-world data and generate business insights.
